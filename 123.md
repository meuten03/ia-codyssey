**스토리보드(기획) 문서**

* 포함 항목: 브랜드 아이덴티티, 캠페인 목표/핵심 메시지, 씬 구성(씬별 필수 필드), 사용 도구 목록

* 씬별 필수 필드(모든 씬 공통):
브랜드 아이덴티티
브랜드명 memorial scent
브랜드 콘셉트 잊고있던 공간의 냄새를 향으로 복원하는 니치 향 브랜드
제품명 Basement memory / rain
브랜드 키워드 노스탤지어/오래된 집/습기 머금은 나무/ 낡은 책/ 포근한 이불/ 조용한 오후/기억의 향
Top Note
젖은 흙, 차가운 공기, 오래된 종이

Middle Note
낡은 나무, 린넨, 비누향, 말린 꽃

Base Note
머스크, 앰버, 파우더리 우드, 은은한 곰팡이 느낌의 어스 노트

주의: “곰팡이 냄새”라고 직접 쓰면 거부감이 들 수 있으니, 어스 노트, 오래된 목재, 습기 어린 공기처럼 표현하는 것이 좋아요.
 
 
 
 씬 번호 / 씬 길이(초)

 캠페인 목표
일상에서 잊고 있던 오래된 집의 기억과 분위기를 향수·디퓨저를 통해 감각적으로 전달한다.
 목표 메시지(한 문장)
 “어떤 냄새는 공간이 아니라 기억을 데려온다.”
 타깃
20~30대 감성 소비자
니치 향수, 인테리어 디퓨저에 관심 있는 사람
레트로, 빈티지, 노스탤지어 감성을 좋아하는 사람

영상 톤앤매너
차분함
빈티지
약간의 습기와 어둠
따뜻한 기억
필름 카메라 느낌
저채도 브라운/그린/베이지 컬러



 화면 구성(구도/피사체/배경/텍스트 유무)
 내레이션(또는 화면 카피)
 사용 도구(이미지/비디오/오디오)와 사용 목적
 입력 프롬프트(원문) + 출력 결과 요약(한 줄)
 생성 결과 파일명 또는 링크 표기(파일명 규칙 자유, 단 일관성 유지)

* 추가 권장: 최소 1개 씬에 대해 “프롬프트 수정 전/후”와 수정 이유를 기록


씬 1
항목	내용
씬 번호 / 씬 길이	Scene 01 / 4초
목표 메시지	오래된 집으로 들어가는 첫 감각을 보여준다.
화면 구성	어두운 지하 계단. 좁은 구도, 낡은 벽지, 먼지 낀 빛. 화면 중앙 아래쪽으로 계단이 이어짐. 텍스트 없음.
내레이션/화면 카피	“문을 열면, 오래된 공기가 먼저 말을 건다.”
사용 도구와 목적	이미지: Midjourney로 지하 계단 이미지 생성 / 비디오: Runway로 천천히 내려가는 카메라 무빙 / 오디오: 낮은 앰비언트 사운드
입력 프롬프트 원문	“old basement stairs in a Korean vintage house, dim warm light, humid air, dust particles, nostalgic mood, cinematic, film photography, muted brown and green tones, no people, 16:9”
출력 결과 요약	낡은 지하 계단과 습기 어린 공기가 느껴지는 이미지 생성

씬 2
항목	내용
씬 번호 / 씬 길이	Scene 02 / 5초
목표 메시지	지하실 냄새를 시각적 분위기로 표현한다.
화면 구성	낡은 나무 선반, 오래된 책, 유리병, 천으로 덮인 상자. 배경은 흐릿하게 처리. 텍스트 없음.
내레이션/화면 카피	“젖은 흙, 낡은 책, 오래된 나무의 조용한 냄새.”
사용 도구와 목적	이미지: Midjourney로 오브젝트 생성 / 비디오: Runway로 먼지 입자와 천천히 줌인 / 오디오: 낮은 공기음과 나무 삐걱거림
입력 프롬프트 원문	“vintage basement shelf with old books, wooden boxes, glass bottles, slightly humid atmosphere, soft dust in the air, nostalgic but beautiful, cinematic close-up, shallow depth of field, 16:9”
출력 결과 요약	지하실 특유의 오래된 사물과 습도감을 표현한 장면

씬 3
항목	내용
씬 번호 / 씬 길이	Scene 03 / 5초
목표 메시지	지하실의 차가운 분위기에서 할머니 집의 따뜻한 기억으로 전환한다.
화면 구성	할머니 집 방. 꽃무늬 이불, 오래된 장롱, 햇빛이 들어오는 창문. 화면 하단에 짧은 카피 삽입.
내레이션/화면 카피	화면 카피: “차갑지만, 이상하게 포근한 기억.”
사용 도구와 목적	이미지: DALL·E로 할머니 방 이미지 생성 / 비디오: Pika로 햇빛이 천천히 움직이는 효과 / 오디오: 잔잔한 피아노 추가
입력 프롬프트 원문	“cozy grandmother’s room in an old Korean house, floral blanket, wooden wardrobe, soft afternoon sunlight through window, warm nostalgic atmosphere, vintage film look, muted beige tones, 16:9”
출력 결과 요약	할머니 집 특유의 따뜻하고 오래된 방 분위기 생성

항목	내용

화면 구성	낡은 책, 나무 조각, 린넨 천, 말린 꽃, 흙빛 유리병이 테이블 위에 놓임. 제품 원료 무드보드처럼 구성.

사용 도구와 목적	이미지: Firefly로 향 노트 오브젝트 이미지 생성 / 비디오: CapCut으로 컷 편집 / 오디오: 부드러운 질감의 배경음악
입력 프롬프트 원문	“perfume moodboard on wooden table, old paper, dried flowers, linen cloth, small wooden pieces, earthy color palette, nostalgic scent ingredients, elegant product photography, soft natural light, 16:9”
출력 결과 요약	제품 향 노트를 시각화한 감성적인 무드보드 이미지
