# LLM 기반 업무 자동화

# 실행 로그

## 1. LLM 실행 시나리오

본 검증 시스템의 실효성을 증명하기 위해 실제 업무 프로세스를 기반으로 한 실행 시나리오를 아래와 같이 정의함.
 항목 | 내용 |
|---|---|
| 업무 과업 | 비정형 고객 클레임 이메일 데이터 기반 'R&D 전달용 기술 이슈 보고서' 자동 정제 및 추출 |
| 사용 모델 | Gemini 3.5 Flash |
| 목적 | Prompt v2를 적용하여 시간 비용 절감, 원인 분석의 정확도 향상 (환각 통제), 부서 간 협업 최적화 |

## 2. 10턴 기술 보고서 정제 및 문맥 유지 검증 로그

* **검증 규칙:** 1턴(Turn)은 [사용자 입력 ➔ AI 출력] 1세트를 기준으로 삼음.
* **복합 시나리오:** 대화 중간 운영 정책 변경(Turn 5) 및 데이터 지연 유입(Turn 7) 조건을 결합하여 모델의 컨텍스트 유지력을 스크리닝함.
